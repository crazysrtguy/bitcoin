# BITCOIN Cult Website

A satirical Bitcoin-themed website called 'Brainwashed Idiots Totally Convinced Of Inevitable Nirvana' (BITCOIN).

## About

This website is a satirical take on Bitcoin and crypto culture, featuring over-the-top interactive elements that poke fun at the sometimes cult-like behavior in the cryptocurrency space.

## Features

- **Bitcoin Lore**: The sacred history of our BITCOIN memecoin
- **Zombie Horde**: Convert normies into Bitcoin zombies
- **Bitcoin Price Justifier**: Generate absurd justifications for any Bitcoin price target
- **Hopium Meter**: Breathe in the hopium and reach enlightenment
- **Diamond Hands Training**: Test your ability to HODL through market crashes
- **Price Prophecy Generator**: Receive increasingly absurd Bitcoin price predictions
- **Cult Initiation**: Join the cult with our sacred rituals
- **How to Buy**: Instructions for purchasing our memecoin on pump.fun

## Technical Details

- Built with React, TypeScript, and Styled Components
- Interactive animations using Framer Motion
- Sound effects with Howler.js

## Disclaimer

This is a satirical website. Nothing here constitutes financial advice. BITCOIN is a fictional memecoin that will launch on pump.fun with a 1 billion supply and no taxes.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
